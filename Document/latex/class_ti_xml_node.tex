\subsection{詳細描述}
The parent class for everything in the Document Object Model. 

(Except for attributes). Nodes have siblings, a parent, and children. A node can be in a document, or stand on its own. The type of a \hyperlink{class_ti_xml_node}{Ti\+Xml\+Node} can be queried, and it can be cast to its more defined type.

Every node in the X\+ML tree will be conditionally visited and the host will be called back via the \hyperlink{class_ti_xml_visitor}{Ti\+Xml\+Visitor} interface.

This is essentially a S\+AX interface for Tiny\+X\+ML. (Note however it doesn\textquotesingle{}t re-\/parse the X\+ML for the callbacks, so the performance of Tiny\+X\+ML is unchanged by using this interface versus any other.)

The interface has been based on ideas from\+:
