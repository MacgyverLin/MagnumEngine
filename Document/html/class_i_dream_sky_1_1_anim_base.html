詳細描述

Base class For all type of Skeletal Animation.

This is the base class for all type of Skeletal Animation. The Animation type can be extended by inheritance.

列舉型態成員說明文件

enum IDreamSky::AnimBase::LoopBehavior

This Enum specified the way the animation is played.

列舉值
ONESHOT 

play until last frame and stop at first frame

CLAMP 

play until last frame and stop at last frame

LOOP 

play until last frame and rewind to first frame, loop forever until stop or pause

PINGPONG 

play until last frame and reverse dir, loop forever until stop or pause
