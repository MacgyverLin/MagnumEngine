詳細描述

RigidBody Component.

RigidBody Component is a base component for handling Physics simulation. Developer can specified Mass, Shape, Position, Velocity, Force, AngularPosition, AngularVelocity, Torque for Rigidbody Static, Kinematic, Dynamic Rigidbody is supported.
